## [Live instance](https://jam-tour-list.netlify.app/)

### Features:

- useState;
- useEffect;
- conditional rendering;
- fetching external data.

### Initializing local instance

```
cd tour-list && npm install && npm start
```

### Building

```
cd tour-list && npm run build
```
